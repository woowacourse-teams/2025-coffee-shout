## 똥손도 즐기는 커피빵 전쟁!

점심시간마다 반복되는 '오늘 커피는 누가 살까?' 고민, 지루하지 않으신가요?

커피빵은 **커피 내기를 간편하고 유쾌하게** 즐길 수 있도록 만든 플랫폼입니다.

단순한 뽑기, 미니 게임 그리고 룰렛 시스템으로 더욱 재밌는 경험을 제공합니다.

👉🏻[게임하러가기](https://coffee-shout.com)
